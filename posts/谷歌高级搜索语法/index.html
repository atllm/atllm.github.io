Google Advanced Search syntax

Google搜索引擎提供了一些高级语法和运算符，可以帮助您更精确地进行搜索。以下是一些常见的高级搜索语法和运算符：

引号

将搜索词或短语放在引号内，以搜索完全匹配的内容。例如，搜索 “纽约时报” 将只返回包含这个短语的结果。

减号

使用减号来排除特定的词语。例如，搜索 苹果 -手机 将返回有关苹果公司的结果，但不包括与手机相关的结果。

星号

使用星号作为通配符来替代任何单词或短语。例如，搜索 人工智能 * 将返回包含以"人工智能"开头的短语的结果。

site

使用 site: 运算符来限制搜索结果到特定网站或域名。例如，搜索 site:yanxizhu.com java8 将只返回yanxizhu.com站点上关于java8的结果。

link

使用 link: 运算符来查找链接到特定网页的其他网页。例如，搜索 link:yanxizhu.com 将返回链接到yanxizhu.com网站的其他网页

related

使用 related: 运算符来查找与特定网站相关的网页。例如，搜索 related:yanxizhu.com 将返回与yanxizhu.com网站相关的网页。

intitle

用于搜索包含特定关键字的标题。例如，intitle:google 将返回所有标题中包含“google”的网页。

intext

使用 intext: 运算符来查找包含特定关键词的网页。例如，搜索 intext:自然风景 摄影 将返回包含"自然风景"关键词的摄影网页。

inurl

用于搜索包含特定关键字的网页。例如，inurl:登录 将返回所有包含“登录”的网页。

filetype

使用 filetype: 运算符来搜索特定文件类型的结果。例如，搜索 filetype:pdf 技术白皮书 将只返回 PDF 文件的技术白皮书。

define

使用 define: 运算符来获取特定词汇的定义。例如，搜索 define:科技 将返回关于"科技"一词的定义。

cache

使用 cache: 运算符来查看 Google 缓存的网页版本。例如，搜索 cache:example.com 将显示示例网站的 Google 缓存版本。

AROUND

使用 AROUND(N) 来搜索两个词或短语之间距离不超过 N 个词的结果。例如，搜索 苹果 AROUND(3) iPhone 将返回包含"苹果"和"iPhone"之间不超过三个词的结果。

-inurl:https：用于查找网站上非 https 的页面。

-site：域名 你的独特内容：用于查找你的网站内容有没有被分享或者剽窃。

name (site:twitter.com site:facebook.com site:linkedln.com youtube.com) 用于快速查找某人的社交档案。前提是你需要知道对方的姓。

这些高级搜索语法和运算符可以帮助您更精确地定位您所需的信息。请记住，Google 可能会随着时间的推移对其搜索算法和语法进行更改，因此有时需要查看 Google 的官方文档以获取最新信息。
